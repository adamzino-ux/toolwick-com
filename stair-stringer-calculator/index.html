How to Use This Stair Calculator

Enter the total rise (floor-to-floor vertical height) and optionally the total run. Adjust the rise and run sliders to find a comfortable step size — the calculator will determine the optimal number of steps and stringer length.

Switch to Advanced to set tread thickness, nosing overhang, stringer lumber width, and number of stringers. Results include full code compliance checks against IRC, IBC, or UK Building Regulations.
